# 本地常驻

本地服务负责飞书长连接、读取本机状态、本地搜索、Ollama embedding 和私有扩展。GitHub Actions 只是本机离线时的兜底。

## 推荐启动

```bash
python -m feishu_companion
```

兼容启动：

```bash
python main.py
```

## LaunchAgent

仓库提供 `launchd/com.example.feishu-companion-bot.plist` 模板。部署时要把路径改成自己的绝对路径。

```bash
mkdir -p ~/Library/LaunchAgents
cp launchd/com.example.feishu-companion-bot.plist ~/Library/LaunchAgents/
launchctl bootstrap gui/$(id -u) ~/Library/LaunchAgents/com.example.feishu-companion-bot.plist
launchctl kickstart -k gui/$(id -u)/com.example.feishu-companion-bot
tail -f bot.log
```

`caffeinate -dimsu` 会尽量阻止系统睡眠，但手动锁屏后仍可能影响前台窗口读取、屏幕状态判断和部分本机 UI 能力。服务本质上是一个普通 macOS 进程。
