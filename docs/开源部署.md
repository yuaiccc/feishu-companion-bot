# 开源部署检查

公开发布前，至少检查这些内容：

- `.env` 没有被跟踪。
- `state.json` 没有被跟踪。
- `memory_data/` 没有被跟踪。
- `bot.log`、`bot.err.log` 没有被跟踪。
- 私有 profile 没有被跟踪。
- 本地扩展 `local_*.py`、`*_private.py` 没有被跟踪。
- 二维码、截图、通话缓存没有被跟踪。
- README、docs、示例 profile 不包含真实姓名以外的敏感信息、真实地址、token、私密聊天内容。

建议命令：

```bash
git status --short
git ls-files | rg '(^\.env$|state\.json|memory_data|\.log$|local_.*\.py|_private\.py|png$)'
git diff --check
```

如果要做通用开源模板，真实关系设定只放在本地未跟踪 profile 或部署环境变量中。
