# 开源部署检查清单

## 必须确保不提交

```gitignore
.env
state.json
*.log
memory_data/
profiles/sange-shushu.json
local_*.py
*_private.py
feishu_login_qr.png
.uploads/
```

## profile 模板化

- 公开仓库只放 `profiles/default.json` 和 `profiles/example-couple.json`
- 真实人设放在 `profiles/my-profile.json`（加入 `.gitignore`）

## 本地扩展

本地私有逻辑通过环境变量加载：

```env
LOCAL_DAILY_JOB_MODULE=local_daily_job
```

扩展文件 `local_daily_job.py` 已在 `.gitignore` 中。

## Dry Run

默认 `DRY_RUN=true`，部署后确认工作正常再改为 `false`。
