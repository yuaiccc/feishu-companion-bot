# 飞书配置

## 创建飞书应用

1. 登录 [飞书开放平台](https://open.feishu.cn/)，创建企业自建应用
2. 获取 `App ID` 和 `App Secret`
3. 在「应用功能」→「机器人」中启用机器人
4. 配置事件订阅：
   - `im.message.receive_v1` — 接收消息
   - `card.action.trigger` — 卡片按钮回调

## 权限配置

在「权限管理」中添加：

```text
im:message
im:message:send_as_bot
im:message:readonly
im:message.reactions:write
im:resource
cardkit:card:write
```

> `cardkit:card:write` 用于 CardKit 流式卡片回复（创建卡片实体、流式更新文本、关闭流式）。不开通时流式回复会自动降级为「发文本消息 + 编辑」。

## 获取必要 ID

- `FEISHU_BOT_OPEN_ID`：机器人自身 Open ID，可在应用详情页查看
- `FEISHU_CHAT_ID`：目标群聊 ID，或用户 Open ID（私聊时）

## 环境变量

```env
FEISHU_APP_ID=cli_xxx
FEISHU_APP_SECRET=xxx
FEISHU_BOT_OPEN_ID=ou_xxx
FEISHU_CHAT_ID=oc_xxx
FEISHU_READ_MESSAGES=true
```

## 卡片消息

卡片使用飞书 Card 2.0 格式：

```json
{
  "schema": "2.0",
  "header": {
    "title": {"tag": "plain_text", "content": "标题"},
    "template": "blue"
  },
  "elements": [...]
}
```

## 流式回复

私聊回复走飞书官方 CardKit 流式卡片接口（打字机效果），流程：

1. `POST /cardkit/v1/cards` 创建 `streaming_mode:true` 的卡片实体，拿到 `card_id`
2. `POST /im/v1/messages` 发送卡片实体（`content` 引用 `card_id`），消息预览显示 `[生成中...]`
3. `PUT /cardkit/v1/cards/{card_id}/elements/{element_id}/content` 多次传入**全量**文本，飞书自动渲染打字机效果
4. `PATCH /cardkit/v1/cards/{card_id}/settings` 设 `streaming_mode:false` 收尾，预览变为正文

依赖 `cardkit:card:write` 权限与卡片 JSON 2.0（飞书客户端 7.20+）。群聊不走流式，直接发文本。
