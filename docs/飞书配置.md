# 飞书配置

这个项目通过飞书自建应用工作，不建议只靠群机器人 webhook。需要机器人被加入目标群聊，并启用相应事件订阅。

## 必需配置

```env
FEISHU_APP_ID=cli_xxx
FEISHU_APP_SECRET=xxx
FEISHU_CHAT_ID=oc_xxx
FEISHU_BOT_OPEN_ID=ou_xxx
FEISHU_READ_MESSAGES=true
```

如果需要给 owner 私聊推送状态或记忆确认：

```env
FEISHU_STATUS_CHAT_ID=oc_p2p_xxx
```

## 常用权限

- `im:message`
- `im:message:send_as_bot`
- `im:resource`
- `im:message.reactions:write`
- `im:message:readonly`

如果接入妙记、文档评论、云文档编辑，需要单独增加对应权限。字段和权限必须以飞书官方文档或 `lark-cli schema` 为准，不要按猜测拼 payload。

## 事件订阅

- 群聊和私聊消息：订阅消息接收事件。
- 卡片按钮：订阅 `card.action.trigger`。

群聊只在 @ 机器人时回复；私聊可以直接回复。
